# 🌍 AI 旅行攻略 · 微信小程序

智能旅行规划小程序，输入目的地、预算和天数，一键生成包含景点、天气、美食、安全提示的全方位旅行攻略。

## ✨ 功能

| 功能 | 说明 |
|------|------|
| 🏛️ **景点推荐** | 目的地必去景点与网红打卡点，附精美示例图片和评分 |
| 🌤️ **天气预报** | 游玩期间逐日天气预报，包含温湿度、风力、紫外线及出行建议 |
| 🍜 **美食推荐** | 性价比分析与环境优美的美食店铺推荐 |
| 🛡️ **境外安全** | 境外目的地自动触发安全提醒，包含当地新闻动态、大使馆联系方式、安全评级 |

## 🚀 快速开始

### 1. 导入微信开发者工具
- 打开微信开发者工具
- 选择「导入项目」→ 选择 `miniprogram` 文件夹
- 填入 AppID（可使用测试号）

### 2. 修改 AppID
在 `project.config.json` 中替换 `appid` 为你的小程序 AppID。

### 3. 编译预览
- 点击「编译」即可在模拟器中预览
- 真机预览请点击「预览」扫码

## 🔌 接入真实数据

当前使用 Mock 数据开发，替换为真实 API：

1. 编辑 `utils/api.js`，取消注释 `wx.request` 部分的代码
2. 填写真实的后端接口地址 `BASE_URL`
3. 编辑 `utils/mock.js` 可按需移除，生产环境使用后端返回数据

**推荐接入的 API：**
- 天气：和风天气 / OpenWeatherMap
- 景点：高德地图 POI 搜索 / 美团
- 新闻：新闻聚合 API
- 安全评级：中国外交部领事司 API
